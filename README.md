# Sovereign Ledger

[![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=flat&logo=Flutter&logoColor=white)](https://flutter.dev)
[![Material 3](https://img.shields.io/badge/Material--3-%236750A4.svg?style=flat&logo=material-design&logoColor=white)](https://m3.material.io)

Sovereign Ledger is a premium, cross-platform financial management application designed for precision and elegance. Built with Flutter, it offers a high-fidelity experience for tracking wealth, managing budgets, and gaining deep fiscal insights across Mobile, Tablet, Desktop, and Web.

---

## Key Features

### Wealth Dashboard
- Liquid Wealth Portfolio: Track your real-time market valuation and portfolio growth.
- Spending Trends: Dynamic charts visualizing your weekly spending velocity.
- Recent Ledger: A quick-access view of your latest transactions with detailed categorization.

### Intelligent Budgeting
- Category Allocations: Set precise limits for Food, Travel, Subscriptions, and more.
- Real-time Progress: Visual progress bars and "Remaining" indicators to keep your spending on track.
- Flexible Management: Add, edit, and adjust budget allocations on the fly.

### Advanced Insights
- Data Visualization: Donut charts and trend analysis to understand your fiscal behavior.
- Monthly Overviews: Deep dives into utilization percentages and overspent categories.
- CSV Export: Take your data anywhere with one-click export functionality (Desktop/Web).

### Security & Experience
- Facial Liveness Check: State-of-the-art biometric verification for mobile security.
- Cross-Platform Navigation: Adaptive layouts that shift from Bottom Nav (Mobile) to a Custom Side Rail (Desktop).
- Keyboard Shortcuts: Power-user control with Ctrl+N for new entries and Ctrl+1-4 for rapid navigation.

## Getting Started

### Prerequisites
- Flutter SDK (v3.22.0 or higher)
- Dart SDK (v3.4.0 or higher)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/expense_tracker_app.git
   cd expense_tracker_app
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the application:
   - Desktop (Windows/macOS/Linux):
     ```bash
     flutter run -d windows # or macos/linux
     ```
   - Web:
     ```bash
     flutter run -d chrome
     ```
   - Mobile:
     ```bash
     flutter run
     ```
